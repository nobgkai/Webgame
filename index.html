เนื้อเรื่องเกม

ผจญภัยไปกับคุณแพนด้า!

Panda's Chinese Quest คือเกมพิมพ์ศัพท์แนว RPG ที่จะพาคุณไปปราบ มอนสเตอร์ทั้งหมด 10 ตัว ด้วยพลังคำศัพท์ภาษาจีน! ทุกครั้งที่ตอบถูก คุณจะโจมตีมอนสเตอร์ได้เต็มแรง แต่ถ้าตอบผิด... เลือด (HP) ของคุณจะลดลง! ฝึกฝนศัพท์ให้แม่น แล้วมาดูกันว่าใครจะรอดถึงบอสตัวสุดท้าย 🐉

ระบบ HP สุดตื่นเต้น
บอส 10 ด่าน
ฝึกพิมพ์พินอิน/จีน

วิธีเล่นเกม

แค่ 3 ขั้นตอนง่าย ๆ ก็ออกไปผจญภัยได้เลย!

1

ฟังและอ่าน

ฟังเสียงและอ่านคำศัพท์จีนที่ปรากฏบนหน้าจอ

2

พิมพ์พินอินหรือตัวอักษร

พิมพ์คำตอบเป็นพินอินหรือตัวอักษรจีนให้ถูกต้อง

3

โจมตีมอนสเตอร์!

ตอบถูก = โจมตีเต็มแรง! ตอบผิด = เสีย HP ระวังด้วยนะ!

เตรียมคีย์บอร์ดภาษาจีน

ก่อนออกไปผจญภัย อย่าลืมติดตั้งคีย์บอร์ดพินอินกันก่อนนะ!

คอมพิวเตอร์ (PC/Mac)

  1. เปิด Settings (การตั้งค่า) ของเครื่อง
  2. ไปที่หมวด Language & Region หรือ Keyboard
  3. เลือก Add Input Method / Keyboard
  4. ค้นหาและเพิ่ม Chinese (Simplified) - Pinyin
  5. สลับคีย์บอร์ดด้วย Win + Space (Windows) หรือ Ctrl + Space (Mac)

มือถือ (iOS/Android)

  1. เปิด Settings (การตั้งค่า) บนมือถือ
  2. ไปที่ General → Keyboard (iOS) หรือ System → Languages & Input (Android)
  3. เลือก Add New Keyboard
  4. ค้นหาและเพิ่ม Chinese - Pinyin
  5. กดปุ่ม 🌐 บนคีย์บอร์ดเพื่อสลับไปใช้พินอินตอนเล่นเกม
